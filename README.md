# buscoderecho.com

Construido con [Svelte](https://svelte.dev) y [TailwindCSS](https://tailwindcss.com)

# Requisitios

Tener instalado [Node.js (LTS)](https://nodejs.org) y [pnpm](https://pnpm.io)

## Comandos

| Comando        | Descripción                                                             |
| :------------- | :---------------------------------------------------------------------- |
| `pnpm install` | Instala las dependencias necesarias.                                    |
| `pnpm dev`     | Inicia el servidor en el modo de desarrollo.                            |
| `pnpm build`   | Compila el sitio web y lo guarda en `./build/`.                         |
| `pnpm preview` | Previsualiza la compilación de forma local.                             |
| `pnpm lint`    | Analiza el código en busca de problemas y verifica el estilo.           |
| `pnpm format`  | Formatea automáticamente el código según las convenciones establecidas. |
